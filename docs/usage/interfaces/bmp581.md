# Датчик кубового давления BMP581

Начиная с версии прошивки 0.2.6.0 в Openconnect в рамках эксперимента вводится поддержка высокоточного датчика давления bmp581.

Датчик так же участвует в работе подсистемы порогов и генерирует события при их превышении

# Схема включения

!!! note "Важная информация"

    Подтягивающие резисторы добавляются на всю шину i2c, а не на каждое подключенное устройство
